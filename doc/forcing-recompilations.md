## Libraries not using pkg-config

To recompile these libs, delete `<libname>.a` with the same name in `/local[32|64]/lib`:

    libdl
    libflite
    libglslang
    libgpac_static
    libmujs
    libpython311
    libshaderc_combined
    libsoxr (sox resampling library only)
    libxavs
    libxvidcore
